# TimeFolio

2025년도 1학기 경북대학교 진로탐색수업 도전 과제 <br><br>

 도전과제: 개인용 포트폴리오 웹사이트 제작 <br>
 - 개인의 기술 스택, 프로젝트 경험, 성장 과정을 체계적으로 정리 <br>
 - 협업자나 채용 담당자에게 자신을 효과적으로 알리는 플랫폼 <br>
